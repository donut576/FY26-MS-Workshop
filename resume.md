## 專案經驗與技術實踐

### 專案一：Talent Swap - 技能交換媒合平台（印象最深刻的專案）
**時間**：2024年9月 - 2024年12月  
**角色**：前端開發與系統整合 | **使用技術**：HTML/CSS/JavaScript, Bootstrap, Django, SQLite3, Gemini 1.5 Flash

**專案背景與目標**：
這是一個「以技能換技能」為核心概念的媒合平台，讓使用者可以用自己擅長的技能交換想學習的技能。目標是打造一個友善、直覺的使用者介面，讓技能交換變得簡單有趣。

**我的職責與貢獻**：
- 負責前端介面設計與實作，使用 Bootstrap 建立響應式網頁
- 與後端團隊協調 API 規格，整合 Django 後端與前端頁面
- 整合 Gemini 1.5 Flash API，實作智能技能推薦功能
- 設計並實作使用者註冊、登入、技能發布與媒合流程

**技術挑戰與解決方案**：
- **挑戰**：如何讓 AI 推薦功能與使用者互動流暢整合
- **分析**：研究 Gemini API 的呼叫方式與回應格式，理解非同步請求的處理機制
- **解決**：使用 JavaScript Fetch API 處理非同步請求，並設計 Loading 動畫提升使用者體驗
- **成果**：成功實作智能推薦功能，使用者可以快速找到適合的技能交換對象

**印象最深刻的經驗**：
最有成就感的是第一次成功串接 AI API 的那一刻。一開始遇到 CORS 問題和 API 回應格式解析困難，我花了很多時間閱讀文件、測試不同的實作方式。最後透過理解瀏覽器的同源政策和 Django 的 CORS 設定，成功解決問題。這個經驗讓我深刻體會到，遇到問題時不能只是複製貼上解決方案，而是要理解背後的原理。
